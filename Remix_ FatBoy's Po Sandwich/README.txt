# README for Remix: FatBoy's Po Sandwich #


* `applab/applab.js` - This file contains the applab library, which has all the
  javascript that applab uses to run this  app.

* `applab/applab.css` - This file contains some default CSS styling for this app.

* `index.html` - This is the html file that I  designed in applab using the
  Design tab. It contains all the buttons, text inputs, and other controls for
  this app.

* `style.css` - This file defines the custom styling for each control in the
  app that I  specified under the Design tab. 

* `code.js` - This file has all the code you wrote I for  my app. 

* `assets` - This is a directory with all the asset files (pictures and sounds)
  that I uploaded in applab.
